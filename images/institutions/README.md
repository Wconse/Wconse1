# Изображения учебных заведений

Поместить в эту папку изображения для вкладки "Учебные заведения".

## Структура имён файлов

Для каждого учебного заведения:

- `{id}-logo.jpg` — логотип (рекомендуемый размер: 120×120 px)
- `{id}-1.jpg`, `{id}-2.jpg`, `{id}-3.jpg` — фото для галереи (рекомендуемый размер: 600×400 px)

## Форматы

Поддерживаются: `.jpg`, `.jpeg`, `.png`, `.webp`

## Пример

Если вы скачали логотип ЮУрГУ как `susu.png`, переименуйте его в `miass_susu-logo.png` и поместите в эту папку.
